# 手工验收执行脚本-转写页时间轴治理门禁-2026-04-03

## 目标

- 在合并前确认时间轴改造未引入交互回归、布局错位和可用性问题。
- 为 PR 提供统一、可追溯的人工验证证据。

## 适用范围

- 触发条件：改动涉及 `src/pages`、`src/components`、`src/styles/pages/transcription-*`、时间轴交互逻辑或样式治理规则。
- 执行频率：每个相关 PR 至少 1 次；发布前主干再执行 1 次全量复核。

## 执行前准备

1. 拉取目标分支并安装依赖。
2. 执行 `npm run gate:timeline-phase1`，确保自动门禁先通过。
3. 启动本地环境 `npm run dev`。
4. 准备 1 份包含文本轨和媒体轨的样例工程。

## 冒烟用例（必须全过）

### A. 时间轴基础布局

1. 进入转写页，确认时间轴首屏无遮挡、无错位、无溢出滚动条。
2. 调整窗口宽度（桌面宽屏、中等宽度、窄宽度），确认关键控件仍可见可点。
3. 打开/关闭左侧或附加面板，确认时间轴宽度与工具栏联动正常。

### B. 轨道与头部交互

1. 折叠/展开轨道分组，确认高度变化平滑且内容不重叠。
2. 拖拽轨道顺序（如支持），确认释放后顺序和渲染一致。
3. 检查轨道头部按钮（静音、显隐、焦点）状态切换是否即时生效。

### C. 时间块与编辑行为

1. 在文本轨执行选中、拖拽、边缘缩放，确认起止时间更新正确。
2. 连续快速编辑 5 次以上，确认无卡死、无明显抖动、无残影。
3. 新增/删除文本片段后，确认相邻片段布局与碰撞处理正常。

### D. 媒体轨与联动

1. 播放/暂停/跳转后，播放头与时间轴标尺同步。
2. 缩放时间轴（放大/缩小）后，片段宽度变化合理，文本可读性保持可用。
3. 若存在翻译/音频联动轨，确认切换语言或音轨后不会出现错轨显示。

### E. 工具栏与观察器（Observer）

1. 触发工具栏中与时间轴相关操作，确认按钮状态、禁用态、提示信息正确。
2. 在页面滚动和面板切换后，确认 portaled 工具栏定位无漂移。
3. 快速重复开关 3 次，确认不会出现重复挂载或样式叠加异常。

## 验收标准

- 所有用例通过，且无 P0/P1 问题。
- 允许存在不阻塞上线的 P2 问题，但必须登记工单与修复计划。
- PR 必须附至少 2 张截图（首屏、编辑后状态）或 1 段短录屏。

## 失败处置

1. 任一 P0/P1 失败：阻塞合并，先修复再复测。
2. 同一问题复测 2 次失败：升级为发布风险并通知负责人。
3. 回归来源不明确：按最近改动范围执行二分回退定位。
