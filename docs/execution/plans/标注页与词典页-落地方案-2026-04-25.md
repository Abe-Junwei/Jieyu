# 标注页与词典页 — 详细落地方案

## 〇、落地总览

```
M1 标注页 MVP（1-2 周）
┌──────────────────────────────────────────────────┐
│ SegmentListView (virtual)                        │
│  ├─ SegmentRow: 原文 / gloss(可编辑) / 翻译        │
│  ├─ AutoGlossPanel: AI 建议 → 确认/拒绝            │
│  ├─ SegmentPlayer: 行内复听                        │
│  └─ LayerTabs: 层切换                               │
└──────────────────────────────────────────────────┘
         │ read canonical units via annotationLaneReadScope
         │ write gloss via LinguisticService
         ▼
    转写页（现有）—— 同一数据库，修改即同步

M3 词典页（1 周）
┌──────────────────────────────────────────────────┐
│ ThreePanelLayout                                  │
│  ├─ 左: LexemeList + minisearch 搜索               │
│  ├─ 中: LexemeEditor（编辑/新建）                    │
│  └─ 右: ConcordancePanel（命中语段）                 │
└──────────────────────────────────────────────────┘
```

**数据获取：**
```typescript
// useAnnotationData.ts
function useAnnotationData(projectId: string, layerId: string) {
  // 1. 通过 annotationLaneReadScope 获取 canonical units
  const units = pickTimelineUnitsForTranscriptionLayer(/*...*/);

  // 2. 获取每个 unit 的 token/morpheme
  const tokens = useQuery({
    queryKey: ['annotation-tokens', projectId, layerId],
    queryFn: () => LinguisticService.listUnitTokens(unitIds),
  });

  return { units, tokens, isLoading, error };
}
```

#### SegmentRow


**交互流：**
```
1. 用户点击行 → 聚焦 gloss 输入框
2. 按 Space → 播放段音频
3. 输入 gloss → 实时 Leipzig 校验
4. 若有 AI 建议 → 幽灵文本显示 → Tab 键接受
5. Ctrl+Enter → 保存 → auto-save debounce 550ms → 触发转写页刷新事件
```

#### AutoGlossSuggestion

```typescript
interface AutoGlossSuggestionProps {
  tokenForm: string;
  suggestion: { gloss: string; confidence: number; sourceLexemeId?: string };
  onAccept: () => void;
  onReject: () => void;
}
```

**显示逻辑：**
1. 调用 `AutoGlossService.matchToken(tokenForm)` → 返回 match + confidence
2. confidence > 0.7 → 幽灵文本（灰色半透明）
3. confidence 0.4-0.7 → 候选列表
4. confidence < 0.4 → 不显示

### 1.3 数据流

```
annotationLaneReadScope.ts
    │ resolveCanonicalUnitForTranscriptionLaneRow()
    ▼
useAnnotationData hook
    │ 返回 { units, tokens, glosses }
    ▼
SegmentRow
    │ onChange → setState (local draft)
    │ onBlur → LeipzigValidator.validateGlossLine()
    │ Ctrl+Enter → LinguisticService.updateTokenGloss(tokenId, gloss, layerId)
    │              → 触发 'jieyu:annotation-gloss-changed' 事件
    ▼
转写页监听 custom event → React Query invalidate(['transcription-units'])
```

### 1.4 AutoGlossService 接入

```
AutoGlossService.matchToken(form)
    │ exact match → LexemeDocType.lemma = form → 返回 gloss + confidence: 1.0
    │ prefix match → LexemeDocType.lemma startsWith form → gloss + confidence: 0.8
    │ substring match → form 包含在 lemma 中 → gloss + confidence: 0.5
    ▼
返回 AutoGlossCandidate { gloss, confidence, lexemeId }
```

### 1.5 键盘快捷键

| 键 | 行为 |
|----|------|
| `↓/↑` | 移到下/上一个 segment |
| `Space` | 播放/暂停当前段音频 |
| `Tab` | 接受 AI 建议 → 填入 gloss 字段 |
| `Enter` | 保存当前行 gloss |
| `Ctrl+Enter` | 保存并移到下一行 |
| `Escape` | 取消编辑，恢复原值 |

### 2.3 useLexiconSearch — minisearch 集成

```typescript
// useLexiconSearch.ts
import MiniSearch from 'minisearch';

function buildLexiconIndex(lexemes: LexemeDocType[]): MiniSearch {
  const ms = new MiniSearch({
    fields: ['lemma', 'gloss', 'definition', 'semanticDomains'],
    storeFields: ['id'],
    searchOptions: {
      boost: { lemma: 2, gloss: 1.5 },
      fuzzy: 0.2,      // 拼写容错
      prefix: true,     // 前缀搜索
    },
  });
  ms.addAll(lexemes.map(formatForIndex));
  return ms;
}
```

### 2.6 导出实现

```typescript
// LexiconExportMenu.tsx
function LexiconExportMenu({ lexemes, format }: { lexemes: LexemeDocType[]; format: 'flex' | 'csv' }) {
  const handleExport = async () => {
    if (format === 'flex') {
      // 复用 FlexService 词条序列化
      const xml = await FlexService.serializeLexemeEntriesAsFlexText(lexemes);
      downloadBlob(xml, `lexicon-export-${Date.now()}.flextext`);
    }
    if (format === 'csv') {
      const csv = lexemes.map(l => [
        l.id,
        formatMultilang(l.lemma),
        formatMultilang(l.senses[0]?.gloss ?? {}),
        l.lexemeType ?? '',
        l.language ?? '',
        l.senses[0]?.category ?? '',
      ].join(',')).join('\n');
      downloadBlob(csv, `lexicon-export-${Date.now()}.csv`);
    }
  };
}
```
