# 语料库（产品定位与执行方案 · 2026-04-28）

**角色**：**站内「语料库」工作台（`/corpus`）当前产品真值**——以 **浏览、检索、集选与管理项目内语料** 为 **主叙事**；**术语 / 田野代码 / 可选缩写备忘（非书目、非参考文献库）** 与语料 **同壳可达**；**轻度短稿** 与 **既有 AI 智能体** 为 **辅**；生成结果与语料须 **一键复制 / 导出**；**版式与长篇真值** 在用户 **既有外软**（Word、Pages、Typora、Obsidian、Overleaf 等）中完成。

**口径说明**：本文“产品真值”用于声明 **一期默认方向**，不等同于“语料库能力已全面开放”。开放结论仍以门槛文档与发布验收记录为准。

**与《[写作页开发路线图-2026-04-22.md](./写作页开发路线图-2026-04-22.md)》关系**：该文件为 **全功能站内写作工作台（二期技术母版）**，YAML **`status: deferred`**。本节为 **一期默认交付形态**（产品对外称 **语料库**）；若实现与本文冲突，**以本文为语料库产品口径**，并回写架构摘录或 ADR。二期恢复后，长编辑 / 四出 / 书级协同等仍以该路线图与 ADR 0010–0016 为索引。

**路由**：主路径 **`/corpus`**；历史书签 **`/writing`** 重定向至 **`/corpus`**（实现以代码为准）。

---

## 一、产品定位与非目标

### 1.1 一句话

**田野调查 / 转写语境下的「语料库 + 术语备忘枢纽」（无书目）**：让用户 **少切换、少复制错、少丢元数据**；**不重做** 通用字处理器。

### 1.2 目标（可验收）

1. **语料库核心**：项目内片段 / 标注 / 对齐结果的可 **检索、筛选、预览、多选、打包元数据**；**复制 / 导出** 带 **可追溯引用**（时间码、`corpusRef` 或等价 id、项目名）。  
2. **术语备忘（非书目）**：与项目绑定的 **词表 / 术语 / 田野代码 / 可选缩写速查** 等 **轻量字段**（**不包含** 参考文献书目；**不在** 语料库维护 `bibliographic_sources`）。若保留缩写能力，可 **可选** 复用 **`abbreviations`**；不需要则 **B 区可收缩为仅术语表/纯文本表**。与语料面板 **同一页或同壳 Tab** 可达。  
3. **轻度写作**：**短文本** 草稿（字段笔记、语例旁注、段落级 md）；**可选** 小型 **CodeMirror 6** 面（**单文档、软上限**，不承诺全书版心；与路线图 [2.8.3](./写作页开发路线图-2026-04-22.md#2-8-可访问性-事故手册-大文档边界与横向收口) **Worker/大文档上限** 叙事同轨）；**权威长文** 默认 **外软**。  
4. **智能体**：**仅** 经既有 **`useAiChat` + `tool_call` + 工具注册`**；窄适配器在路线图称 **`writingBridgeAdapter`**（与 `writingDocAdapter` **子集或别名**）；实现 PR 可命名为 **`corpusBridgeAdapter`**，语义须与其 **子集** 一致；产出 **先审后复制**，**默认不落盘** 到转写主链。  
5. **外软桥接**：**多 MIME 剪贴板**（`text/plain` 底线 + `text/html` / md 策略）+ **可选小 bundle**（`snippets.md` + `manifest.json` + README）；错误码与 i18n 与路线图 [2.7.1](./写作页开发路线图-2026-04-22.md#2-7-体验-安全-迁移与生态补充) **对齐**。

### 1.3 非目标（一期不承诺）

- **站内** 四出 golden、全书 vault、G4 写作协同、Pandoc 预览真值 **不作为** 语料库 go 门槛（见 **deferred** 路线图）。  
- **参考文献书目 / `bibliographic_sources` 工作流**：**不在** 语料库一期范围（若仓库已有类型，**不**在 `/corpus` 挂载编辑或导出入口）。  
- **不**追平 Word / Pages **版式引擎**；不与 Obsidian **插件宇宙**对标。  
- **不**以第二文档模型（PM 等）承载正文真源（[路线图 · 第三节](./写作页开发路线图-2026-04-22.md#三-技术架构-四层-唯一主图) 精神仍适用：**站内短稿仍是字符串真源**）。

### 1.4 与标注页、词典页：防重复边界

语言学 **标注 / 项目词典** 真值见 **[标注页与词典页开发路线图-2026-04-22.md](./标注页与词典页开发路线图-2026-04-22.md)**（含 **SSOT**、**FLEx 导出**、**对象级 AI 采纳**）。语料库一期 **不替代** 该路线图，仅与之 **并列**。

| 维度 | **语料库一期（本文）** | **标注页 / 词典页** | **防重复规则** |
|------|------------------------|----------------------|----------------|
| **B 区「术语备忘」** | **术语备忘 / 可选缩写**（**无书目**；**不含** `bibliographic_sources`） | **`lexemes` / 义项 / 词形 / 附件**、**`token_lexeme_links`** | **禁止**在语料库做 **词条 CRUD、入词典冲突分流、FLEx 词典包导出**；需编词条 **跳转词典页** |
| **A 区「语料」** | **多选、预览、元数据、出站**（只读为主） | **分层编辑、写库、多实例层** | **禁止**在语料库复制标注页 **层实例** UI；改 POS/gloss/morph **跳转标注页或转写页** |
| **AI** | **段落级场记 / 论文向生成**；`writingBridgeAdapter` / `corpusBridgeAdapter`；**复制优先** | **对象绑定 suggestion**；`annotationAdapters`；**采纳后写库** | **工具注册表分区**；语料库 **默认不** 调用「逐 token 采纳写库」路径 |
| **导出** | **剪贴板 MIME、外软稿、bundle** | **FLEx / flextext 词典导出** | **入口与文案分轨** |

补充：词典页当前“浏览骨架”已纳入开放态统计（基础骨架口径），但词典产品级开放结论仍按门槛文档 `go/no-go` 规则判定。

## 二、信息架构（页面分区 · 实现 PR 可微调命名）

| 分区 | 用户任务 | 主要组件 / 数据来源（口径） |
|------|-----------|------------------------------|
| **A · 语料** | 按层/话者/时间/标签筛选；波形或列表预览；多选 | 转写宿主已有 **selection / segment** 模型；**只读** 为主，写回须走既有事务边界 |
| **B · 术语与备忘** | 搜词、改注、管田野代码/术语表、**可选**缩写速查 | **无书目**；不读写 `bibliographic_sources`；若用 `abbreviations` 须在 PR **写明可选**；其余用 **轻量表或本地 session**，**与导出/AI 工具共享类型** 时优先 **合同包或 `src/db/types`**，避免第三套字段名 |
| **C · 轻度写作** | 草稿框、语例旁注、大纲 bullet | **CM6 可选**；默认 **软上限**（字数或行数，与路线图 [2.8.3](./写作页开发路线图-2026-04-22.md#2-8-可访问性-事故手册-大文档边界与横向收口) 叙事同轨）；**自动保存** 策略须写清（Dexie 轻表或 session） |
| **D · 智能体** | 分析、对比、扩写、生成段落 | 既有对话壳 + **上下文注入**（当前选区 + 术语条目 + 可选草稿）；**分级提示**（严格引用 / 润色 / 自由） |
| **E · 出站到外软** | 复制为 Word / MD / HTML；下载 bundle | 与 **第五节** MIME 矩阵一致；**错误码 + i18n**（[2.7.1](./写作页开发路线图-2026-04-22.md#2-7-体验-安全-迁移与生态补充)） |

**导航**：`/corpus` 下 **Tab 或侧栏** 切换 A–E；**移动端** 若不做，须在 i18n 或门槛中 **显式降级**。

## 三、典型用户路径

| 步骤 | 用户动作 | 系统应做到 |
|------|----------|------------|
| 1 | 在 **A** 选中多条语料 | 右侧或底部 **预览拼接**；展示 **元数据** |
| 2 | 在 **B** 核对术语 / 备忘（**无书目**） | 与语料引用 **同一屏** 可对照 |
| 3 | 在 **C** 写几句衔接稿 | **软上限** 告警；**可选** 自动保存 |
| 4 | 在 **D** 让智能体 **摘要 / 扩写 / 对比** | 上下文含 **A 选区 + B 条目**；结果 **可编辑** |
| 5 | 在 **E** 选 **「复制为 Word」** 或 **导出 bundle** | 外软 **粘贴即用**；失败 **可诊断** |

## 四、剪贴板与目标软件矩阵（须实测迭代）

**原则**：**`text/plain` 底线**；**`text/html`** 服务 Word/Pages；**md** 服务 Typora/Obsidian；**LaTeX** 仅 **纯文本片段**。

| 目标类 | MIME / 策略 | 验收要点 |
|--------|---------------|----------|
| **Word / Pages** | `text/html` + `text/plain`；可选 **制表符语例** | Unicode 音标保留；复杂结构 **降级** 须在 UI 标明 |
| **Typora / Obsidian** | `text/plain` 为主 + 可选 `text/markdown` | GFM 差异须提示 |
| **Overleaf** | 纯文本 LaTeX 模板（宏 **白名单** 另表） | **不**在站内编译 |
| **通用** | 仅 `text/plain` | 任何环境可用 |

**安全**：生成 HTML 须 **白名单标签**；与转写 **粘贴消毒** 是否共用，须在实现 PR **点名模块**。

## 五、分期（P0–P3，与 deferred 路线图 G **解耦**）

| 阶段 | 交付 | 验收 |
|------|------|------|
| **P0** | **A + E**：语料多选 → 纯文本 / md 复制 + 元数据；**无 AI** 可闭环 | Word + 一款 md 编辑器 **fixture ≥3** |
| **P1** | **B** 术语浏览与导出；**HTML 剪贴板**；错误码 | 与 P0 矩阵 + **空选 / 超长** 诊断 |
| **P2** | **D** 智能体接入 + **C** 轻度写作（最小 CM6 或 textarea） | **不**静默写转写；工具 **可审计** |
| **P3（可选）** | bundle 导出；与 deferred 路线图 **G1.5 四出** **可选对接** | 单独里程碑 + ADR 一句 |

**开放门槛**：建议为 **「语料库」** 在 [门槛文档](./独立工作台开放门槛与复用边界-2026-03-30.md) 中 **单列 go 条件**（小于全写作工作台）；`layoutGuard` **双分支** 仍须（flag off ≡ 现占位）。

## 六、与既有 AI 栈（不写新智能体）

- **适配器**：路线图 **`writingBridgeAdapter`**；实现可名 **`corpusBridgeAdapter`** — **只读语料工具 + 生成文本工具**；返回 **字符串** 供 **E** 区复制。  
- **工具示例**：`fetch_corpus_snippet`、`list_selection_metadata`、`expand_for_fieldnotes`、`glossary_draft`…（以合同与评审为准）  
- **会话**：优先 **`corpusSessionId`**（或沿用路线图 **`writingSessionId`** 命名）与转写会话 **隔离**（若尚未实现，在 P2 PR 中 **首次引入** 并写 ADR 一句）。
