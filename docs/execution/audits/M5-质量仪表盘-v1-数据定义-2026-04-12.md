# M5 质量仪表盘 v1 数据定义（2026-04-12）

## 1. 目标维度
- version：构建版本（例如 `1.0.0`）
- module：功能模块（例如 `transcription` / `ai-chat` / `shell`）
- environment：运行环境（例如 `local` / `ci` / `prod`）

## 2. 指标项（v1）
1. `ux.web_vitals.lcp_ms`
- 类型：histogram
- 单位：ms
- 说明：首屏可感知渲染延迟。

2. `business.transcription.segment_action_latency_ms`
- 类型：histogram
- 单位：ms
- SLO（P95）：<= 300ms
- 说明：切分/合并/保存语段关键操作耗时。

3. `business.e2e.main_path_success_rate`
- 类型：ratio
- 单位：ratio
- SLO：>= 0.99
- 说明：主路径 E2E 成功率。

4. `ai.chat.first_token_latency_ms`
- 类型：histogram
- 单位：ms
- SLO（P95）：<= 8000ms
- 说明：AI 首包延迟。

5. `ai.chat.completion_success_count`
- 类型：counter
- 单位：count
- 说明：AI 完成轮次计数。

## 3. 汇总口径
- count：样本数
- p50：50 分位
- p95：95 分位
- min / max：极值

## 4. go/no-go 首版判定
- P0：指标目录或测试失败，直接 no-go。
- P1：指标定义存在但业务未接入，允许灰度并告警。
- P2：仅展示层细节问题，进入优化队列。
