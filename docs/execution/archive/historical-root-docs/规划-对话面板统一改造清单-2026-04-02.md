# 对话面板统一改造清单（2026-04-02）

## 目标
- 将项目内对话面板统一到“新建项目”面板风格。
- 统一基础结构：`overlay -> card -> header/body/footer`。
- 统一视觉 token：遮罩、圆角、边框、阴影、输入控件紧凑度。

## 已纳入统一样式体系的面板
- `ProjectSetupDialog`
- `AudioImportDialog`
- `ConfirmDeleteDialog`
- `DeleteLayerConfirmDialog`
- `SpeakerActionDialog`
- `ShortcutsPanel`
- `LayerActionPopover`
- `SidePaneActionModal`
- `BatchOperationPanel`
- `TimelineLaneHeader`（锁定说话人弹层）
- `LeftRailProjectHub`（项目导入弹层）
- `VoiceAgentWidget`（确认/消歧 compact 弹层）
- `NotePopover`（compact 弹层）

## 本轮统一改造范围
- 全局浮层卡片基准样式（`floating-panel`）统一到项目对话卡片风格。
- 浮层遮罩（`layer-action-popover-backdrop`）统一为标准遮罩色与层级。
- 快捷键面板遮罩（`shortcuts-panel-backdrop`）统一到标准遮罩。
- 项目导入遮罩（`left-rail-project-import-backdrop`）统一到标准遮罩。
- 常用对话组件补齐 `role="dialog"` / `aria-modal="true"`。

## 后续可选增强
- 将内联提示文本样式抽离为 `dialog-notice-*` 原子类，减少组件内联样式。
- 增加 `dialog-card-danger` / `dialog-card-compact` 变体 token，覆盖删除确认等高风险弹窗。
- 为全部对话面板补齐一致的 `aria-labelledby` 和标题 id 绑定。

- 运行态层操作弹窗统一以 `LayerActionPopover` 为准。
- 历史组件 `LayerManagerPopover` 已删除，不再作为任何生产入口存在。
- 共享层动作标签与弹窗专属文案已拆分为 [src/i18n/layerActionLabels.ts](../../../../src/i18n/layerActionLabels.ts) 与 [src/i18n/layerActionPopoverMessages.ts](../../../../src/i18n/layerActionPopoverMessages.ts)，避免继续沿用 manager 命名。
