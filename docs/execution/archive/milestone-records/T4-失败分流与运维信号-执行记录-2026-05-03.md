## 4. 史诗 §7「T4」退出条件对齐说明

Release evidence JSON 根字段 **`toolDecisionFailureSignals`** 同时提供多类可消费聚合信号，例如：

- **`failureSignals.triageCounts`**（运维四象限分布）
- **`failureSignals.partialExecutionProgressRows`** 与 **`failureSignals.rollbackErrorCountBuckets`**（propose 部分失败与 rollback 错误计数分桶）
- **`durableHandoff.humanInterventionRate` / `handoffReasons`**（与 `durableOrchestration` 交叉）

另：`evidenceIndex` 含 **`t4.tool-decision-failure-signals.v1`** 条目。
