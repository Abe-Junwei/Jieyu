## 已知风险
1. 仍是“并行模型”阶段
- 旧表仍在写入与保留；未执行“完全切读 + 停旧写 + 旧表裁剪”。

2. 外部导入复杂场景
- 极端 tier 结构下，bridge link 的语义仍建议通过真实样例再做人工验收。

3. Phase-4 blocker 已明确
- `segment_links` 需要补齐 `sourceLayerId` / `targetLayerId` 索引，避免层级删除与巡检依赖全表扫描。
- utterance embeddings 需要最小失效机制，保证默认转写文本变化或语段删除后不会继续命中陈旧向量。
