## 快速判定规则
- 任一 P0 失败：禁止发布。
- 任一 P1 未修复：进入有条件发布评审（需明确回滚预案和修复计划）。
- P2 可带缺陷发布，但需记录在发布说明与后续迭代计划。

## 回滚触发条件（建议）
- 上线后出现“翻译层大面积空白/错读旧值”。
- 删除 layer 或 utterance 导致页面异常或数据不一致。
- AI 检索/RAG 命中明显回退到旧文本。

## 回滚动作（建议）
1. 立即关闭 `segmentBoundaryV2Enabled`。
2. 保留现场快照与日志，导出受影响项目数据。
3. 按 R-01~R-12 对应 CASE 复现并定位。
