# 发布说明：声学 Wave E 产品化收口（2026-04-09）

## 版本范围
- 模块：acoustic provider contract、AcousticAnalysisService、AI 分析面板 provider 配置区、assistant context/controller 透传
- 目标：完成 Wave E 在当前阶段的产品化收口（配置、可达性检测、失败回退、状态可见化）

## 主要变更
1. Provider 契约与运行时配置收口
- 新增 runtime config 归一化与本地持久化能力（routing strategy、enabled、endpoint、apiKey、timeout）
- 新增外部 provider 健康检查结果分型：disabled、unconfigured、401、403、timeout、network/http/unknown
- 保持 provider-neutral 契约，默认本地链路稳定可用

2. 外部 provider 受控试接入链路稳定化
- external 可达且启用时尝试外部分析
- external 失败时自动回退 local，不阻断分析主流程
- 缓存键按 provider + endpoint 隔离，避免不同外部端点污染

3. 面板交互与状态可视化补齐
- AI acoustic tab 新增 provider 配置 UI（routing/enabled/endpoint/apiKey/timeout）
- 新增保存配置、重载配置、健康检查操作
- 新增健康状态、延迟、鉴权提示与 fallback 状态展示
- 保存后支持立即刷新 provider 状态，不依赖下一次分析触发

4. 上下文链路贯通
- ReadyWorkspace -> assistant controller/context -> AiPanel 的刷新回调与 provider 状态透传已贯通
