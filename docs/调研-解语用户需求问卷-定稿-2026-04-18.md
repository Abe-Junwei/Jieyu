# 第二部分：副卷甲（已实现功能 · 矩阵量表）

## 副卷甲导语（说明文字）

下列为解语**当前已提供**的能力簇。请按您的**真实使用**评分。

**未使用或不了解该行**，请在各列统一选择「**不适用**」（或在问卷星中为该行勾选「本条跳过」类选项，与产品约定一致即可）。

### 矩阵列定义（列标题 · 每列 1～5 + 不适用）


| 列名       | 含义                                |
| -------- | --------------------------------- |
| **实用性**  | 对您工作流程的实际帮助：1 无帮助 … 5 帮助很大        |
| **需求性**  | 对该能力的需求强度：1 不需要 … 5 非常需要          |
| **满意度**  | 仅在使用过时评价：1 很不满意 … 5 很满意；未使用选「不适用」 |
| **保留意愿** | 是否希望长期保留：1 可移除 … 5 必须保留           |


### 矩阵行（共 18 行 · 每行一条「评价对象」）

1. 转写时间轴与语段边界编辑
2. 波形监听与播放头同步
3. 录音采集与媒体导入替换
4. 语段切分合并与删除
5. 批量时间偏移缩放等操作
6. 多层转写与翻译对照视图
7. 图层排序约束与依赖提示
8. 范围选择与批量操作面板
9. 说话人管理与指派
10. 全局搜索替换（含正则等）
11. 键盘快捷键与 JKL 穿梭
12. 撤销重做与自动保存恢复
13. 逻辑时间轴纯文本轨与状态提示
14. PDF 文献并排预览
15. 语言元数据与正字法桥接工作台
16. 词典浏览与检索（只读）
17. 外部格式互操作与项目快照归档
18. AI 对话 RAG 引用与语音智能体及协作云

**录入备注：**

- 第 17 行涵盖 ELAN、TextGrid、FLEx、Toolbox、Transcriber 及 JYM 等项目归档；若只用过其中一部分，按整体印象评分即可。  
- 第 18 行将 **AI 侧边栏、RAG 脚注、语音智能体、协作云** 合并评价；若需拆细，可复制本页为两页矩阵分别调查「AI 与语音」与「协作云」。
